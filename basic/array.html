二維陣列/多維陣列

什麼是二維陣列

陣列其實就像是一個火車，火車有以下性質

  1. 火車是從0開始
  2. 火車車廂內可以裝任何東西（但所有車廂需要同一類型ex: int, float, double, 陣列!!）

其實從2.就可以發現，在陣列裡面塞一個陣列其實是可以的

先來看一下一維陣列

一維陣列 Type ar[3]

二維陣列 int arr[3][8]

其實二維陣列就只是一維陣列裡面的每一項都是一維陣列，所以當你取了arr[1]，之後，把他當成一個int [8]的陣列就可以了。
