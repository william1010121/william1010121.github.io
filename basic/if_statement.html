條件判斷式(if statements)

語法

if 語法跟for語法是c++中最重要的兩個語法，現在先來看if語法

布林值邏輯運算子

if成立條件是看if( ... )內的條件出來是否=0如果是0 的話就不成立，其他都是成立的

if(-1)是成立的喔
請玩玩看以下程式碼

假設有兩個bool值a, b, true(1)代表成立,false(0)代表不成立

  1. && (and) 成立條件: a跟b都要為true
  2. || (or) 成立條件:a跟b至少有一個為true
  3. ^ (xor) 成立條件:a,b一個true一個false
  4. ! (not) : 如字面意思，true false,falsetrue

打程式碼的時候可以直接打if( a == b ), if( a >= b )
以下的運算符號都成立

  1. ==(等於), !=（不等於）
  2. >, >=
  3. <, <=
