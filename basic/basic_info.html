就像每個語言都有各自的語法一樣，c++也有自己的語法

先來看一個基本的程式碼

可以看到

  • 第一行有一個#include<...>
  • 所以的操作都在int main內
  • 每一行都有; 結束
  • 一些特別的字元有 std::, ex:std::cin, std::cout
  • 注釋使用的是//(單行)和/* */(多行)

這邊就有一個問題了

a 是什麼?

a 是一個變量，在第三行中可以看到他是一個int的變量。
有關於int的內容，會在之後提到

#include是什麼？

#include可以引入由其他人幫我們寫好的程式碼，站在巨人的肩膀上來使用

std:: 是什麼，std::大部分#incldue進來的函數前面都需要加上，你可以把他想成地址，#include進來的都在std這個地方內，如果以後想要自己做屬於自己的函式庫，你就可以實作在mycode::這個地方。

但是為了方便我們通常使用 using namespace std;讓我們可以不用每次都輸入std::
因此如上的程式碼會長這樣
